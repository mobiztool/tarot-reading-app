# Story 4.8: Onboarding Tutorial Enhancement

## Status

**Approved**

## Story

**As a** first-time user, **I want** interactive tutorial, **so that** I feel confident using the app.

## Acceptance Criteria

1. Interactive tutorial: tooltip-based walkthrough
2. 5 steps max: Welcome, Choose type, Select cards, Discover, Save/share
3. Tooltips highlight relevant UI
4. Skip option always visible
5. Progress: dots (step 2/5)
6. Replay: option in settings
7. Completion message
8. Analytics: completion rate, drop-off
9. Adaptive: different for logged-in vs anonymous
10. Mobile optimized
