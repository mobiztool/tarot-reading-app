# Story 3.6: Encyclopedia Search & Filter Enhancement

## Status

**Approved**

## Story

**As a** user, **I want** advanced search and filtering, **so that** I can quickly find specific cards.

## Acceptance Criteria

1. Real-time search (debounced)
2. Search matches: name, keywords, meanings
3. Highlighted results
4. Filter by suit
5. Filter by element
6. Filter by card type (Court, Numbered, Major)
7. Sort options
8. URL persistence (shareable filters)
9. Clear filters button
10. Results count
11. Empty state with suggestions
12. Mobile: collapsible filter panel
13. Performance: instant filtering

## Tasks / Subtasks

- [ ] **Task 1: Search Implementation** (AC: 1, 2, 3)
  - [ ] Search input with debounce (500ms)
  - [ ] Filter cards by text match
  - [ ] Highlight matching terms
- [ ] **Task 2: Filter UI** (AC: 4, 5, 6)
  - [ ] Filter buttons/dropdowns
  - [ ] Multiple filter support
- [ ] **Task 3: URL State** (AC: 8)
  - [ ] Use query params
  - [ ] Shareable filter URLs
- [ ] **Task 4: Mobile Filters** (AC: 12)
  - [ ] Bottom sheet or modal
  - [ ] Touch-optimized
- [ ] **Task 5: Testing** (AC: 1-13)
  - [ ] Test search accuracy
  - [ ] Test filter combinations
