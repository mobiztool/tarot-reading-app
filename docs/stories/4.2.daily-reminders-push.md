## Story

**As a** user, **I want** daily reminders to do my tarot reading, **so that** I build consistent practice.

2. Time picker: select reminder time
3. Timezone handling: local timezone
4. PWA push notifications: request permission, send at time

10. Graceful degradation: in-app reminder if push not supported
11. Easy unsubscribe
12. Email fallback
