# Story 3.4: Tarot Encyclopedia - Card Database Page

## Status

**Approved**

## Story

**As a** user interested in learning about tarot, **I want** to browse all 78 tarot cards and their meanings, **so that** I can deepen my understanding.

## Acceptance Criteria

1. Encyclopedia page (`/cards`) with all 78 cards
2. Grid layout: responsive (3-4 per row desktop, 2 mobile)
3. Each card: image, name (Thai + English)
4. Organized by suit: Major Arcana, Wands, Cups, Swords, Pentacles
5. Tab navigation: switch between suits
6. Clickable thumbnails: navigate to card detail
7. Major Arcana highlighted: special styling
8. Lazy loading: images load on scroll
9. Search: filter cards by name
10. Filter: All, Major, Minor, by suit
11. SEO: "Tarot Card Encyclopedia | 78 Cards"
12. Breadcrumbs: Home > Encyclopedia
13. Mobile friendly: touch-optimized
14. Loading: skeleton screens
