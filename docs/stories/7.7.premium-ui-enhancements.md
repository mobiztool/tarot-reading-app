## Status

## Story
**As a** premium user, **I want** special UI treatment, **so that** I feel the premium experience.

## Acceptance Criteria
1. Premium badge displayed throughout app
2. Special animations for premium spreads
3. Premium color scheme: gold accents
4. Exclusive card backs for premium users (optional)
5. "Premium" label on restricted content
6. Premium user profile highlighting
7. No ads for premium users (if ads exist)
8. Premium loading animations (more polished)
9. Premium welcome message after subscribe
10. Feels premium: attention to detail
