# Story 4.3: Question Input & Contextualized Readings

## Status

**Approved**

## Story

**As a** user, **I want** to ask a specific question before reading, **so that** interpretation feels relevant.

## Acceptance Criteria

1. Question input on reading selection
2. Placeholder: "What guidance do you seek?"
3. Character limit: 500 chars with counter
4. Saved with reading
5. Shown in result and history
6. AI interpretation ready (foundation)
7. Privacy: only visible to user
8. Question examples/suggestions
9. Always optional
10. Editable in history
11. Analytics: % with questions, themes
