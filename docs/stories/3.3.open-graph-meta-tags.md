# Story 3.3: Open Graph & Social Media Meta Tags

## Status

**Approved**

## Story

**As a** marketer, **I want** proper social media meta tags on all pages, **so that** shared links display beautifully with images and descriptions.

## Acceptance Criteria

1. OG tags on all pages: og:title, og:description, og:image, og:url, og:type
2. Twitter Card tags: twitter:card, twitter:title, twitter:description, twitter:image
3. Homepage: attractive hero OG image
4. Reading results: dynamic OG image based on cards
5. Encyclopedia: unique OG image per card
6. Image specs: 1200Ã—630px, <1MB, CDN hosted
7. Meta descriptions: 150-160 chars, SEO-optimized
8. Facebook Debugger: validates successfully
9. Twitter Validator: cards display correctly
10. Dynamic meta tags: Next.js Metadata API
11. Fallback image: default branded image
