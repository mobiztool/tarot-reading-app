# Story 5.5: Enhanced Signup Value Proposition

## Status
**Ready for Review**

## Story
**As a** guest user, **I want** to see what I get by signing up, **so that** I'm motivated to create an account.

## Acceptance Criteria
1. Landing page updated: show "5 spreads available" (2 guest + 3 login)
2. Signup page enhanced: list 3 login-exclusive spreads
3. Spread preview cards: Love, Career, Yes/No with icons
4. "Unlock 3 more spreads" messaging prominent
5. Reading selection page: show locked spreads with "Login to unlock"
6. Login gate: clear value proposition before signup form
7. Post-signup: celebrate unlocked spreads
8. Analytics: track which spread drove signup
9. A/B test ready: different messaging variations
10. Mobile: value prop clear and thumb-friendly
