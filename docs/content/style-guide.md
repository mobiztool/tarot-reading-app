# Content Style Guide - ไพ่ยิปซี Tarot Reading App

## 1. ภาษาและโทน

### 1.1 โทนเสียง (Tone of Voice)
- **อบอุ่นและเข้าถึงได้** - เหมือนเพื่อนที่เข้าใจและพร้อมช่วยเหลือ
- **ลึกลับแต่ไม่น่ากลัว** - สร้างบรรยากาศเวทมนตร์โดยไม่ทำให้ผู้ใช้กังวล
- **เป็นกลาง** - ไม่ตัดสินว่าดีหรือร้าย ไพ่ทุกใบมีแง่มุมที่หลากหลาย
- **ให้กำลังใจ** - แม้ไพ่ที่ดูท้าทาย ก็มีคำแนะนำเชิงบวก

### 1.2 การใช้คำสรรพนาม
- ใช้ "คุณ" เพื่อพูดกับผู้ใช้โดยตรง
- หลีกเลี่ยง "เรา" ยกเว้นในบริบทที่เหมาะสม
- ใช้คำกลางๆ ไม่ระบุเพศ

### 1.3 ความยาว
- **ชื่อไพ่**: สั้น กระชับ (2-4 คำ)
- **คีย์เวิร์ด**: 3-5 คำ ต่อทิศทาง
- **ความหมายหลัก**: 2-3 ประโยค
- **รายละเอียด**: 1-2 ย่อหน้า

## 2. โครงสร้างเนื้อหาไพ่

### 2.1 ส่วนประกอบที่จำเป็น

```
- ชื่อไพ่ (ภาษาไทย)
- ชื่อไพ่ (ภาษาอังกฤษ)
- หมายเลข (สำหรับ Major Arcana)
- ชุด (สำหรับ Minor Arcana)
- คีย์เวิร์ด (หงาย)
- คีย์เวิร์ด (กลับหัว)
- ความหมายหงาย
- ความหมายกลับหัว
- คำแนะนำ
- ความรักและความสัมพันธ์
- การงานและการเงิน
- สุขภาพ
```

### 2.2 ตัวอย่างรูปแบบ

**ชื่อ**: คนบ้า (The Fool)

**คีย์เวิร์ด (หงาย)**: การเริ่มต้นใหม่, ความไร้เดียงสา, การผจญภัย, ความเชื่อมั่น, ศักยภาพไร้ขีดจำกัด

**ความหมายหงาย**: ไพ่คนบ้าแสดงถึงการเริ่มต้นใหม่ที่เต็มไปด้วยความหวังและความตื่นเต้น คุณกำลังยืนอยู่ที่จุดเริ่มต้นของการเดินทางใหม่...

## 3. หลักการเขียนความหมาย

### 3.1 สิ่งที่ควรทำ ✅
- อธิบายสัญลักษณ์ในภาพไพ่
- เชื่อมโยงกับชีวิตประจำวัน
- ให้คำแนะนำที่นำไปใช้ได้จริง
- กล่าวถึงทั้งแง่บวกและแง่ที่ต้องระวัง
- ใช้ภาษาที่เข้าใจง่าย

### 3.2 สิ่งที่ควรหลีกเลี่ยง ❌
- การทำนายแบบตายตัว ("คุณจะ...")
- การตัดสินว่าดีหรือร้ายเด็ดขาด
- ข้อความที่สร้างความกังวล/กลัว
- ศัพท์เทคนิคที่เข้าใจยาก
- การคัดลอกจากแหล่งอื่น

### 3.3 การเขียนความหมายกลับหัว
- ไม่ใช่ความหมายตรงข้ามเสมอไป
- อาจหมายถึงพลังงานที่ถูกบล็อก
- หรือบทเรียนภายในที่ต้องทำงาน
- ใช้โทนที่สร้างสรรค์ ไม่ทำให้กังวล

## 4. SEO Guidelines

### 4.1 คีย์เวิร์ดหลัก
- ไพ่ยิปซี + [ชื่อไพ่]
- ความหมาย + [ชื่อไพ่]
- ไพ่ทาโรต์ + [ชื่อไพ่]
- [ชื่อไพ่] + หงาย/กลับหัว

### 4.2 การใช้คีย์เวิร์ด
- ใช้คีย์เวิร์ดอย่างเป็นธรรมชาติ
- ไม่ยัดคีย์เวิร์ดมากเกินไป
- รวมคีย์เวิร์ดใน heading และ first paragraph
- ใช้ synonyms และ related terms

## 5. Quality Checklist

### 5.1 ก่อนเผยแพร่
- [ ] ตรวจสอบความถูกต้องของชื่อไพ่
- [ ] ตรวจไวยากรณ์และการสะกดคำ
- [ ] ตรวจความครบถ้วนของทุกส่วน
- [ ] ตรวจความสอดคล้องกับไพ่อื่นๆ
- [ ] ตรวจว่าไม่มีการคัดลอก
- [ ] ตรวจ SEO keywords

### 5.2 หลังเผยแพร่
- [ ] ตรวจสอบการแสดงผลบนเว็บ
- [ ] ตรวจ links ที่เกี่ยวข้อง
- [ ] รวบรวม feedback จากผู้ใช้

## 6. แหล่งอ้างอิง

เพื่อความถูกต้องของเนื้อหา ควรอ้างอิงจาก:
- หนังสือไพ่ทาโรต์แบบดั้งเดิม (Rider-Waite tradition)
- ตำราไพ่ยิปซีที่น่าเชื่อถือ
- ผู้เชี่ยวชาญด้านไพ่ทาโรต์ที่มีประสบการณ์
